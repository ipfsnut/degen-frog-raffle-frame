# Frog.castAction

## Import

```tsx twoslash
import { Frog } from 'frog'
```

## Usage

```tsx twoslash
/** @jsxImportSource frog/jsx */
// ---cut---
import { Button, Frog } from 'frog'

const app = new Frog()

app.castAction('/', (c) => { // [!code focus]
  console.log('Apple and Banana')
  return c.res({ message: 'Action Succeeded' }) // [!code focus]
}) // [!code focus]
```

## Parameters

### path

- **Type:** `string`

Path of the route.

[Read more.](/concepts/routing)

```tsx twoslash
/** @jsxImportSource frog/jsx */
// ---cut---
import { Button, Frog } from 'frog'

const app = new Frog()

app.castAction(
  '/foo/bar', // [!code focus]
  (c) => {
    console.log('Apple and Banana')
    return c.res({ message: 'Action Succeeded' }) // [!code focus]
  }
) 
```

### handler

- **Type:** `(c: Context) => CastActionResponse`

Handler function for the route.

```tsx twoslash

// ---cut---

```
